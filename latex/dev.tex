Người dùng có thể nói chuyện trực tiếp bằng tiếng Việt, hệ thống sẽ tự động nhận diện, suy luận bằng Agentic RAG và trả lời lại bằng giọng nói tự nhiên với độ trễ cực thấp dưới 1 giây.
}
\end{frame}

\section{Kết luận \& Hướng phát triển}
\subsection{Kết luận của đồ án}

\begin{frame}{Kết luận của đồ án}
\begin{itemize}
\item \textbf{Về mặt sản phẩm:}
\begin{itemize}
\item Xây dựng thành công hệ thống trợ lý pháp luật vi dịch vụ hoàn chỉnh, dễ mở rộng.
\item Tích hợp Agentic RAG (LangGraph) trả lời chính xác, dẫn nguồn minh bạch.
\item Hỗ trợ đàm thoại giọng nói thời gian thực, xử lý tài liệu và thanh toán VIP.
\end{itemize}
\item \textbf{Về mặt công nghệ:}
\begin{itemize}
\item Làm chủ các giải pháp: Kong API Gateway, gRPC, Kafka, RabbitMQ.
\item CSDL đa dạng: Postgres, Vector DB (Milvus/Qdrant/Pinecone), Cloudflare R2.
\item Vận hành chuyên nghiệp: Docker, Kubernetes, GitOps (ArgoCD), Terraform (IaC).
\end{itemize}
\end{itemize}

\note{
Sau thời gian nghiên cứu và phát triển, đồ án đã hoàn thành đầy đủ các mục tiêu đề ra.
Về mặt sản phẩm, chúng ta đã xây dựng thành công hệ thống vi dịch vụ trợ lý pháp luật hoàn chỉnh, tích hợp Agentic RAG giúp trả lời chính xác, đàm thoại giọng nói thời gian thực và tự động hóa thanh toán.
Về mặt công nghệ, đồ án đã làm chủ và tích hợp nhiều công nghệ hiện đại từ thiết kế hệ thống DDD, Kong API Gateway, giao tiếp gRPC/Kafka đến việc triển khai hạ tầng tự động bằng Terraform, Kubernetes và GitOps với ArgoCD.
}
\end{frame}

\subsection{Hướng phát triển tương lai}

\begin{frame}{Hướng phát triển trong tương lai}
\begin{itemize}
\item \textbf{Tối ưu hóa AI \& RAG:}
\begin{itemize}
\item Huấn luyện tinh chỉnh (Fine-tune) LLM trên dữ liệu pháp luật Việt Nam.
\item Áp dụng khung đánh giá RAGAS để đo lường và kiểm soát chất lượng phản hồi.
\end{itemize}
\item \textbf{Nâng cấp hạ tầng \& vận hành:}
\begin{itemize}
\item Tích hợp Apache Airflow để điều phối và giám sát tự động đường ống dữ liệu.
\item Triển khai đa cụm Kubernetes và thiết lập hệ thống giám sát tập trung (Prometheus, Grafana, ELK).
\end{itemize}
\item \textbf{Mở rộng tính năng \& phần mềm:}
\begin{itemize}
\item Tối ưu hóa Voice Bot giảm độ trễ đàm thoại; mở rộng cổng thanh toán Stripe.
\item Triển khai kiểm thử tự động toàn diện bằng Playwright.
\end{itemize}
\end{itemize}

\note{
Để hoàn thiện ứng dụng trong tương lai, em tập trung vào các hướng phát triển chính:
Thứ nhất, nâng cao chất lượng AI bằng cách tinh chỉnh LLM chuyên sâu về luật Việt Nam và dùng RAGAS để đánh giá tự động.
Thứ hai, nâng cấp hạ tầng bằng cách tích hợp Airflow điều phối dữ liệu, chạy đa cụm Kubernetes và cấu hình hệ thống giám sát tập trung Prometheus, Grafana, ELK.
Cuối cùng là mở rộng các tính năng nghiệp vụ như thanh toán Stripe, giảm độ trễ Voice Bot và viết kịch bản test tự động bằng Playwright.
}
\end{frame}
