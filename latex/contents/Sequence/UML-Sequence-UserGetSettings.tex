\paragraph{Sơ đồ tuần tự luồng lấy thông tin cài đặt}

Luồng nghiệp vụ này mô tả
quy trình Web Frontend
truy vấn danh sách cài đặt cá nhân của người dùng
từ Setting Service (Neon API) thông qua API Gateway.

\subparagraph{Mô tả tổng quan}

\begin{itemize}

\item \textbf{Yêu cầu lấy cài đặt:}
Người dùng truy cập vào trang cấu hình cài đặt.
Giao diện gửi yêu cầu lấy thông tin cài đặt qua cổng API
tới dịch vụ cấu hình cài đặt kèm theo mã phiên đăng nhập.

\item \textbf{Phản hồi cấu hình:}
Dịch vụ cấu hình cài đặt truy vấn danh sách các cài đặt cá nhân
của người dùng dưới dạng các cặp khóa
và giá trị từ CSDL cài đặt và phản hồi về giao diện để hiển thị cho người dùng.
\end{itemize}
