\begin{frame}{Kết luận của đồ án}
\begin{itemize}
\item \textbf{Về mặt sản phẩm:}
\begin{itemize}
\item Xây dựng hệ thống trợ lý pháp luật hoàn chỉnh, dễ mở rộng.
\item Tích hợp LangGraph trả lời chính xác, dẫn nguồn minh bạch.
% \item TODO: bao nhiều agent node
\item Hỗ trợ đàm thoại giọng nói, xử lý tài liệu và thanh toán VIP.
\end{itemize}
\item \textbf{Về mặt công nghệ:}
\begin{itemize}

\item Tích hợp các giải pháp: Kong API Gateway, gRPC, Kafka, RabbitMQ
\item CSDL đa dạng: Postgres, Vector DB (Milvus/Qdrant/Pinecone)
\item Object Storage: Cloudflare R2, Supabase Bucket
\item Triển khai tự động: Docker, K8s, GitOps, Terraform
\end{itemize}

\end{itemize}

\note{
Sau thời gian nghiên cứu và phát triển, đồ án đã hoàn thành đầy đủ các mục tiêu đề ra.
Về mặt sản phẩm, chúng ta đã xây dựng thành công hệ thống vi dịch vụ trợ lý pháp luật hoàn chỉnh, tích hợp Agentic RAG giúp trả lời chính xác, đàm thoại giọng nói và tự động hóa thanh toán.
Về mặt công nghệ, đồ án đã làm chủ và tích hợp nhiều công nghệ hiện đại từ thiết kế hệ thống DDD, Kong API Gateway, giao tiếp gRPC/Kafka đến việc triển khai
hạ tầng tự động bằng Terraform, Kubernetes và GitOps với ArgoCD.
}
\end{frame}
