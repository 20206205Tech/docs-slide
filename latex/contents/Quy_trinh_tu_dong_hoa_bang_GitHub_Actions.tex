\note{
Sơ đồ quy trình CI/CD tự động trong dự án sử dụng GitHub Actions.
Dự án kết hợp giữa runner miễn phí của GitHub và self-hosted runner trên VPS của riêng mình. Khi có sự thay đổi mã nguồn, hệ thống tự động chạy các kiểm thử (unit test), phân tích chất lượng mã qua SonarQube, sau đó đóng gói mã nguồn thành Docker Image, đẩy lên Docker Hub và tự động cập nhật kho lưu trữ GitOps để ArgoCD triển khai lên cụm Kubernetes.
